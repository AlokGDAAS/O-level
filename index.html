Introduction to web designing


What is Internet?


The Internet is a giant global network of computers


What is WWW (World Wide Web)?


The WWW is the collection of websites stored on the Internet.

Web browsers like Chrome, Firefox, or Edge help you enter this library and open web pages using URLs


What is Website?


A website is a group of related web pages under one domain name.

Examples: google.com, wikipedia.org, amazon.in.


Working of Website?


When you type a website name:

Your browser asks the server for the page.

The server finds the page files (HTML, CSS, JS).

The server sends them back through the Internet.

Your browser reads those files and displays the page.


Webpages


A webpage is a single digital page inside a website.:

The homepage is the main door. Other pages are like different sections of the same shop.

Webpages are built using HTML, styled with CSS, and powered with JavaScript.


Front End


It’s the “face” of the website.

The front end is everything the user sees and interacts with .


Back End


The back end works behind the scenes on the server.

It handles data, security, user login, databases, payments, and logic.

If the front end is the face, the back end is the brain.


Client and Server Scripting Languages


Client-side languages run in the browser.

They control visuals and interactions.


Server-side languages run on the server.

They manage data, login, security, database queries.

Examples: PHP, Python, Node.js, Java.


Responsive Web Designing


Responsive design means your website adjusts itself to different screen sizes.
Phone, tablet, laptop, big monitor, all should show a clean usable layout.
HTML + CSS (especially media queries) are used to make layouts flexible.

It’s like water filling any container shape smoothly.

Types of Websites: Static and Dynamic


Static Website


Built using only HTML, CSS, basic JS

Content stays the same unless changed manually

Fast, simple, good for portfolios or small info sites


Dynamic Website


Changes content automatically

Uses databases and server-side languages

Used for e-commerce, social media, dashboards
