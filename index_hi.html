वेब डिज़ाइनिंग का परिचय


Internet क्या है?


Internet दुनिया भर के कंप्यूटर्स का एक विशाल नेटवर्क है।


WWW (World Wide Web) क्या है?


WWW इंटरनेट पर मौजूद वेबसाइटों का संग्रह है।

Chrome, Firefox या Edge जैसे ब्राउज़र आपको इस लाइब्रेरी में ले जाते हैं और URL की मदद से वेबपेज खोलते हैं।


Website क्या है?


Website एक ही डोमेन नाम के अंदर कई वेब पेजों का समूह होती है।

उदाहरण: google.com, wikipedia.org, amazon.in


Website कैसे काम करती है?


जब आप ब्राउज़र में किसी वेबसाइट का नाम लिखते हैं:

आपका ब्राउज़र सर्वर से वेबपेज मांगता है।

सर्वर HTML, CSS, JS जैसी फाइलें खोजता है।

सर्वर उन्हें इंटरनेट के द्वारा आपके ब्राउज़र को भेजता है।

ब्राउज़र इन फाइलों को पढ़कर आपको वेबपेज दिखाता है।


Webpages


Webpage किसी वेबसाइट का एक डिजिटल पेज होता है।

Homepage मुख्य दरवाज़ा होता है, बाकी पेज दुकान के अलग-अलग सेक्शन जैसे होते हैं।

Webpages HTML से बनाए जाते हैं, CSS से सजाए जाते हैं, और JavaScript से इंटरएक्टिव बनाए जाते हैं।


Front End


यह वेबसाइट का वो हिस्सा है जिसे यूज़र देखता और इस्तेमाल करता है।

फ्रंट एंड में UI, डिज़ाइन और इंटरएक्शन शामिल होते हैं।


Back End


Back End सर्वर पर काम करता है।

यह डेटा, सिक्योरिटी, लॉगिन, डेटाबेस और वेबसाइट की लॉजिक संभालता है।

अगर फ्रंट एंड चेहरा है, तो बैक एंड दिमाग है।


Client और Server Scripting Languages


Client-side भाषाएँ ब्राउज़र में चलती हैं।

ये visuals और interactions को नियंत्रित करती हैं।


Server-side भाषाएँ सर्वर पर चलती हैं।

ये डेटा, लॉगिन, सुरक्षा और डेटाबेस को संभालती हैं।

उदाहरण: PHP, Python, Node.js, Java


Responsive Web Designing


Responsive design का मतलब है कि वेबसाइट हर स्क्रीन साइज पर
अपने आप एडजस्ट हो जाए।
फोन, टैबलेट, लैपटॉप या बड़ा मॉनिटर, हर जगह लेआउट साफ और usable रहे।

इसमें HTML + CSS (media queries) का उपयोग होता है।

यह बिल्कुल पानी की तरह है जो किसी भी बर्तन का आकार ले लेता है।

Websites के प्रकार: Static और Dynamic


Static Website


केवल HTML, CSS और बेसिक JavaScript से बनाई जाती है।

Content तब तक नहीं बदलता जब तक manually edit न किया जाए।

तेज़, सरल, पोर्टफोलियो और छोटे info साइट्स के लिए सही।


Dynamic Website


अपना कंटेन्ट अपने आप बदलती रहती है।

डेटाबेस और server-side languages का उपयोग करती है।

ई-कॉमर्स, सोशल मीडिया और डैशबोर्ड में उपयोग होती है।
