What is HTML?

  1. HTML stands for Hyper Text Markup Language
  2. HTML is the standard markup language for creating Web pages
  3. The DOCTYPE html declaration defines that this document is an HTML5 document
  4. The html element is the root element of an HTML page
  5. The head element contains meta information about the HTML page
  6. The title element specifies a title for the HTML page
  7. The body element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  8. The h1 element defines a large heading
  9. The p element defines a paragraph
